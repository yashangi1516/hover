# ABOUT THE TASK
ANIMATION FOR THE LAUNDRY SERVICES


# PROJECT OVERVIEW
 In this project i have created a replicate the hover animation shown in the reference video for the Call-To-Action (CTA) button in the hero section.

# REQUIREMENT FOT THE PROJECT

When the user hovers over the CTA button:

The button should increase in size.
The button should appear slightly tilted.
The animation should be smooth and visually appealing.

 # FEATURES
I HAVE USED FOLLOWING FEATURES:-

Responsive navigation bar
Desktop navigation links
Mobile hamburger menu
Side menu slides from the right
No JavaScript required
Uses CSS media queries and pseudo-classes
Uses Transform, transition for the animation
